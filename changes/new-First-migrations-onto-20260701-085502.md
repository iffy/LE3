First migrations onto the OCCT kernel (#86, behind `occt` feature): simple extrusions build real BREP prisms tessellated by OCCT, and multi-extrusion bodies fuse into one true unioned solid via an OCCT boolean (both fall back to the hand-rolled mesher for cases the kernel doesn't yet cover — slanted targets, edge chamfers/fillets). Kernel FFI gained solid construction, boolean (fuse/cut/common), volume, and tessellation. OCCT is dead-stripped at function granularity (-ffunction-sections + -dead_strip/--gc-sections) so only used code paths land in the binary.
